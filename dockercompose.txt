Aplicação tem um
- Backend
- Frontend
O frontend comunica com o backend que comunica com o BD

O Docker Compose é um arquivo que informa as partes que estão dentro da Aplicação

docker-compose up // Cria as images e os containers da Aplicação

Yaml
- Data serialization
Linguagem utilizada para escrever configurações

docker-compose.yml (exemplo)

version:"3.8"
services:
    frontend:
        depends_on:
            - backend
        build:./frontend
        ports:
            - 3000:3000
    backend:
        depends_on:
            - db
        build:./backend
        ports:
            - 3001:3001
        environment:
            DB_URL: mongodb://db/vidly
        command: ./docker-entrypoint.sh
    db:
        image: mongo:3.0-xenial
        ports:
            - 27017:21017
        volumes:
            - vidly:/data/db
volumes:
    vidly
